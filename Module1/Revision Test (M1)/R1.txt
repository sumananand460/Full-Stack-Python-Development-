11. Difference between list and tuple

(List): 

- List is mutable (Means elements in list can be changed)
- Consumes more memory
- Slower iteration speed due to dynamic size
- Uses [] brackets

(Tuple):

- Tuple is immutable (Means elements in tuple cannot be changed)
- Consumes less memory
- Higher iteration speed due to fixed size
- Uses () brackets

15. Explain break and continue with an example

=> Both break and continue are loop control statements 

- break is used to terminate the current loop immediately and moves to the next statement after the loop.
Ex: 
for number in range(1, 6):
    if number == 3:
        print(f"Found {number}, breaking loop")
        break
    print(f"Processing number: {number}")
# Output: 1, 2, and "Found 3" message

- continue is used to break the current iteration and continue the next iteration
Ex:
for number in range(1, 6):
    if number == 3:
        print(f"Skipping {number}")
        continue
    print(f"Processing number: {number}")
# Output: 1, 2, "Skipping 3", 4, 5
